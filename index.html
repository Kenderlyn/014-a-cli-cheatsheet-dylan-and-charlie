Terminal Cheat Sheet

Here's a list of some basic Terminal commands that you'll probably be using quite a lot:

  • pwd is the command used to print the current folder/directory you are working in.
  • hostname displays the network name of your computer.
  • mkdir is used to make a folder/directory.
  • cd changes the user into a different directory.
  • ls lists the contents of the directory the user is currently in.
  • rmdir removes a directory of your choosing.
  • pushd stores a directory for later use. It stacks these directories on top of each other with the first directory on the bottom. The order can be changed around by issuing the pushd command another time.
  • popd works with to change your current directory to the one on top of the "stack".
  • cp copies a file or directory.
  • mv is used to move a file or directory.
  • less is typically used to display larger files as it allows you to scroll up and down by line using the k and j keys respectively, and by page using b and f.
  • cat is used to display smaller files as the command itself simply displays the text contained within the file on the Terminal window.
  • xargs executes command-line arguments.
  • find quite literally finds files.
  • grep can be used to find things within files.
  • man allows you to read a manual page of your choosing.
  • apropos lets you find which manual page is appropriate.
