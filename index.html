Command Lists
  • pwd= print working directory
  • ls= list diretory
  • open= opens the file
  • cd= home directory
  • mkdir= make new directory folder
  • pushd= go to a new location
  • popd= return to saved location
  • touch= make an empty file
  • mv= move a file from one location to another
  • rm= how to remove a file
  • clear= Clears the screen
  • cat= Concatenate to screen
  • reset= Resets the terminal display
  • cd ../../= Move 2 levels up
  • less= Output content delivered in screensize chunks
  • top= Displays active processes
  • cmp= Compare two files
  • ln= Make links between files
